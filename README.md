ringo-tabetter
========

Twitterにて、"[リンゴ]"タグ付きでつぶやいたリンゴ名を、集計・HighChartsで表示するアプリです。



開発環境
----------

* Ubuntu 12.04 (Windows7上のVirtualBox内で動作)
* Ruby 1.9.3
* Padrino 0.11.0
* PostgreSQL 9.1.8



セットアップ
----------

1.  api_key.example.yaml をapi_key.yamlにリネームし、TwitterのユーザーIDと各APIキーを記述します。
2.  Herokuへとpushします。
